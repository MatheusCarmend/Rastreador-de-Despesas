Rastreador-de-Despesas
Rastreador para controle simples de despesas, permitindo registrar, visualizar e acompanhar o saldo financeiro diretamente pelo terminal.
📌 Funcionalidades
Adicionar entradas (receitas ou despesas)
Listar todas as entradas
Mostrar saldo atual
Salvar dados em arquivo (expenses.txt)
🛠️ Tecnologias
C#
.NET (Console)
